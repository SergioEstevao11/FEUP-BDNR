# Network Of Thrones
---
Visualization of kinship and conflicts in the European Monarchy, using JanusGraph.
## Execution:

### 1. Start the containers:
In ```./src/```, run:

```bash
docker-compose up
```

### 2. Run load_data.py:
In ```./src/app/backend/app/```, run:
```bash
python3 load_data.py
```
